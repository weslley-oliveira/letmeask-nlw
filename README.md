## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/weslley-oliveira/letmeask-nlw.git && cd letmeask-nlw
```

Follow the steps below
```bash
# Install the dependencies
$ yarn install

# Start the project
$ yarn start
```
